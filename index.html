How to use

Quick Start

Type !squirt followed by your search term to quickly search on different sites.

how to ... !t3 → Start a new T3 Chat AI chat
!g house of cards → Google search for "house of cards"
!yt music → YouTube search for music
!w javascript → Wikipedia article on JavaScript
cats !gi → Google Images search for cats
hello !gt @es → Google Translate "hello" to Spanish
